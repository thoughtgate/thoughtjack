# Building the Docs Site

The ThoughtJack documentation site uses [Docusaurus](https://docusaurus.io/) and follows the [Diataxis](https://diataxis.fr/) framework — tutorials, how-to guides, reference, and explanation.

:::info
Attack scenarios and their documentation now live on [oatf.dev](https://oatf.dev). This site covers ThoughtJack usage, configuration, and architecture.
:::

## Build the docs site

Install dependencies and build:

```bash
cd docs-site
npm install
npm run build
```

## Preview locally

```bash
cd docs-site
npm start
```

This starts a local development server at `http://localhost:3000/`.
