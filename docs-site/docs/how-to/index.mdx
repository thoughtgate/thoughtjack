# How-To Guides

Goal-oriented recipes for specific tasks. Each guide assumes you already have ThoughtJack installed and have run at least one scenario.

| Guide | Goal |
|-------|------|
| [Add a Scenario to the Library](./add-scenario) | Create and register a new attack scenario |
| [Configure Delivery Behaviors](./configure-behaviors) | Set up slow-loris, unbounded-line, and other delivery modes |
| [Configure Side Effects](./configure-side-effects) | Set up notification floods, pipe deadlocks, and other side effects |
| [Use Payload Generators](./use-generators) | Generate nested JSON, garbage, unicode, and other payloads |
| [Run Over HTTP Transport](./http-transport) | Use HTTP/SSE instead of stdio |
| [Integrate with CI/CD](./ci-integration) | Run ThoughtJack in CI pipelines with machine-readable output |
| [Use Dynamic Response Handlers](./dynamic-responses) | Configure `$handler` with HTTP, command, and sequence handlers |
